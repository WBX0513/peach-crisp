# peach-crisp

A new food item has been added to Minecraft: Peach Crisp.

Update log:

**1.1-snapshot-2**：
- Fixed the bug in the "Balanced Diet" challenge, which is now displayed normally.

**1.1-snapshot-1**：
- When obtaining the pink flower cluster, the Peach Crisp will be displayed in the recipe book.

**1.0**：
- Upon obtaining Peach Crisp, the progress "Peach Crisp!" is achieved, which is inherited from "Agriculture and Animal Husbandry".
- Add peach crisp to the "balanced diet" category. (There is a bug that prevents this challenge from being displayed.)

**1.0-snapshot-2**：
- Optimized the texture mapping.

**1.0-snapshot-1**：
- Add peach crisp, and combine two peach cakes with four pink flower clusters and one cookie.
- Consuming it will grant 3 seconds of Health Regeneration II.
